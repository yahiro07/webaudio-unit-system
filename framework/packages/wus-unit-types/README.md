# wus-unit-types

Type definitions for the units used in [webaudio-unit-system](https://github.com/yahiro07/webaudio-unit-system).

This is a type-only lightweight package that is used when implementing units in TypeScript.

If you want to create an unit and do not need host related features, this package is enough.

If you making a host, the host internally included this package and you don't need to install this separated.
